# Landing page — image checklist

Every image slot on `index.html` is a labelled dashed placeholder (it shows the
section, what to shoot and the size right on the page). This is the same list in
one place so you can capture them all at once.

Drop each screenshot in and replace the matching `.ph` placeholder block with an
`<img src="img/your-file.jpg" alt="…">`. Suggested folder: `Doc/landing/img/`.

## Tips
- Export at **2×** (retina) for crispness, then let CSS scale them down.
- Keep a consistent browser chrome / device frame for the front-end shots.
- JPG for photos/screenshots, PNG where you need sharp UI edges or transparency.
- Brand colours used on the page: `#fd4621 → #ff8a3d`; font: Inter.
